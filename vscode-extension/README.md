# AI Serviter VS Code Scaffold

This is a minimal VS Code extension scaffold.

## Development

```bash
cd vscode-extension
npm install
npm run compile
```

Make sure the Python package is installed in your environment:

```bash
cd ../python
python -m pip install -e ".[dev]"
```

Commands:

- `AI Serviter: Analyze Workspace`
- `AI Serviter: Plan Coding Task`
